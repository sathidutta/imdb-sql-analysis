# 🎬 IMDb Data Analysis using SQL

## 📌 Overview
This project focuses on analyzing IMDb movie data using SQL to extract meaningful insights from structured datasets. It demonstrates how relational data can be used to uncover trends in movie ratings, genres, and performance.

---

## 🎯 Objective
The main objectives of this project are:
- Analyze movie dataset using SQL queries  
- Understand relationships between different tables  
- Extract insights to support data-driven decisions  

---

## 🛠 Tools & Technologies
- SQL  
- Excel  
- Relational Database Concepts  

---

## 🗂 Dataset Description
The dataset contains:
- Movie details  
- Ratings  
- Genres  
- Other related attributes  

An Entity Relationship Diagram (ERD) was used to understand table relationships and structure.

---

## 🧠 Key Concepts Used
- Joins (INNER JOIN, LEFT JOIN)  
- Aggregations (COUNT, AVG, SUM)  
- GROUP BY and HAVING  
- Subqueries  

---

## 📊 Analysis Performed
- Extracted top-rated movies  
- Analyzed genre-wise performance  
- Identified trends in ratings and popularity  
- Explored relationships between different attributes  

---

## 🔍 Key Insights
- Certain genres consistently receive higher ratings  
- Popular movies tend to cluster within specific categories  
- SQL queries enable efficient analysis of structured data
